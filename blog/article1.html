10 خصائص CSS ستغير طريقة برمجتك للأبد

كود بالعربي
مطوّر ويب
5 يناير 2025 5 دقائق 980 قراءة

كثير مطورين يكتبون CSS بنفس الطريقة من سنوات — margin، padding، flex، ويفكرون إنهم يعرفون كل شيء. لكن CSS فيها خصائص قوية جداً يتجاهلها الأغلب، ولما تكتشفها تتساءل كيف كنت تشتغل بدونها!

في هذا المقال، 10 خصائص حقيقية مع أمثلة كود جاهزة تقدر تنسخها مباشرة.

ملاحظة
كل الخصائص هنا مدعومة في المتصفحات الحديثة بنسبة 95%+. يمكنك التحقق دائماً على caniuse.com.

1 clamp() — حجم خط ذكي بدون media queries

بدل ما تكتب font-size مختلف لكل breakpoint، clamp() تخلّي الحجم يكبر ويصغر تلقائياً بين حد أدنى وحد أقصى.

CSS 
/* الطريقة القديمة */
h1 { font-size: 24px; }
@media (min-width: 768px) {
  h1 { font-size: 36px; }
}

/* الطريقة الذكية بـ clamp() ✨ */
h1 {
  font-size: clamp(24px, 5vw, 48px);
}

2 gap — المسافة بين العناصر بدون margin

بدل ما تكتب margin-bottom على كل عنصر، استخدم gap مباشرة على الـ container. يشتغل مع flex و grid.

CSS 
.cards-container {
  display: flex;
  flex-direction: column;
  gap: 16px;
}

.grid {
  display: grid;
  gap: 24px 16px; /* row-gap column-gap */
}

3 aspect-ratio — نسبة أبعاد ثابتة بدون هاك

قديماً كنا نستخدم الـ padding-bottom hack. الآن خاصية aspect-ratio تحل المشكلة بسطر واحد.

CSS 
.video-wrapper {
  aspect-ratio: 16 / 9;
  width: 100%;
}

.avatar {
  width: 100%;
  aspect-ratio: 1 / 1;
  border-radius: 50%;
}

4 Logical Properties — CSS يدعم العربية تلقائياً

لو موقعك يدعم RTL وLTR، الـ Logical Properties تتكيّف تلقائياً بدون ما تكتب شيء إضافي.

CSS 
/* Logical — يتكيّف مع RTL/LTR تلقائياً ✨ */
.btn {
  padding-inline: 16px;
  margin-inline-start: 8px;
  border-inline-start: 3px solid blue;
}

5 CSS Custom Properties — متغيرات حقيقية في CSS

CSS 
:root {
  --color-primary: #6B4FBB;
  --radius: 12px;
}

.btn {
  background: var(--color-primary);
  border-radius: var(--radius);
}

/* Dark mode بسطرين فقط! */
[data-theme="dark"] {
  --color-primary: #9B7FE8;
}

6 :has() — "Parent Selector" اللي انتظرناه 10 سنوات

CSS 
.card:has(img) {
  padding-top: 0;
}

.form-group:has(input:focus) label {
  color: var(--color-primary);
  font-weight: 700;
}

7 grid-template-columns: repeat(auto-fill, minmax())

CSS 
.cards-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 24px;
}

8 place-items: center — توسيط بسطر واحد

CSS 
.center-me {
  display: grid;
  place-items: center;
  min-height: 100vh;
}

9 scroll-snap — سلايدر بدون JavaScript

CSS 
.slider {
  display: flex;
  overflow-x: auto;
  scroll-snap-type: x mandatory;
  gap: 16px;
  scrollbar-width: none;
}

.slide {
  flex-shrink: 0;
  width: 100%;
  scroll-snap-align: start;
}

10 Container Queries — الـ Responsive الحقيقي

CSS 
.card-wrapper {
  container-type: inline-size;
  container-name: card;
}

@container card (min-width: 400px) {
  .card {
    display: flex;
    flex-direction: row;
  }
}

الخلاصة

هذه الخصائص العشر ليست مجرد اختصارات — هي طريقة تفكير مختلفة في كتابة CSS. ابدأ بـ clamp() و gap و aspect-ratio وستلاحظ الفرق فوراً.
